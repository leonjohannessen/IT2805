Answers to exercise 3 questions:

  1. Elements positioned with position: fixed are positioned relative to the browser window (viewport). Elements positioned with position: absolute; are on the other hand positioned relative to the next parent element with relative or absolute positioning. If no such element exist the element will be placed based on the page itself.
  2. To select the third element within another element, we can use the :nth-child(An+B) pseudo class, as in the example below where we select the third element of the example list using .example-list > :nth-child(3)
