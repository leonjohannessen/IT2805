Answers to exercise 2 questions:

  1. Inline styles have higher specificity than styles defined in the style tag or in external stylesheets (it can, however, be overridden with the !important-flag).
  2. IDs should be used for unique elements that occur only once. Classes, on the other hand, should be used for styling elements that occur many times on a page according to the 'Don't Repeat Yourself'-principle (DRY). For example, a header element should be referenced to with an id, but sidebar links with related articles should use classes.
  3. RGBA means 'Red, Green, Blue, Alpha'. You can express any color in the RGB color space (with transparency) on the form color: rgba(r,g,b,a); where each value is an int in the range 0-255.
  4. Styles are not content, by definition the link element is metadata, and we therefore include it in the head of the html document. (Linking to external style sheets also allow for multiple pages to share the same style sheet.)
  5. The article p selector would match all the three paragraphs inside the article.

Answers to the questions:

  1. The server provides responses to the requests made by the clients over the HTTP protocol. Usually these responses are in text form, eg. a HTML document.
  2. The client requests data from the server, recieve and process it, and usually renders it to the screen of the client.
  3. An Internet Protocol (IP) address is the unique address of a node in a network. An IPv4 addresses is on the form x.x.x.x, where x can be any integer in the range 0-255.
  4. We use domain names as aliases for IP addresses. Domain Name System (DNS) servers serves as a "phone book" of domain names and their corresponding IP addresses.
  5. World Wide Web ("WWW"/"The Web") is only one of many applications that can be accessed through the Internet. Other applications that use other protocols are for instance E-mail, IRC, and SSH. The Web itself is a distributed system of hypertext documents, where one can jump between documents using hyperlinks.
  6. HyperText Markup Language (HTML) is a set of tags that, together with text, describe the content of web documents.
